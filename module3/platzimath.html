Platzimath

Esta herramienta permite calcular el promedio y mediana de un arreglo numérico. También se puede obtener la moda de un arreglo que contenga cualquier tipo de dato. Y finalmente, como reto se propone una herramienta para obtener la media cuadrática de un arreglo de valores.

Promedio (media aritmética) y mediana

Se debe ingresar los valores numéricos de un arreglo separados por espacio. Por ejemplo, si se desea calcular el arreglo [1, 2, 3, 4] en el siguiente cuadro de texto se debe ingresar "1 2 3 4" sin las comillas y los valores separados por un solo espacio. Si se requiere ingresar un valor decimal (Float o Punto flotante) se debe ingresar el punto flotante como punto (1.2) y no cómo coma (1,2).

Media cuadrática (RMS)

Se debe ingresar los valores de un arreglo separados por espacio. Por ejemplo, si se desea calcular el arreglo [0.94, -0.42, 0.51, 1.29, -0.95] en el siguiente cuadro de texto se debe ingresar "0.94 -0.42 0.51 1.29 -0.95" sin las comillas y los valores separados por un solo espacio. Si se requiere ingresar un valor decimal (Float o Punto flotante) se debe ingresar el punto flotante como punto (1.2) y no cómo coma (1,2).
